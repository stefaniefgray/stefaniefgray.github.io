Papel - Single page portfolio theme **Papel** is a modern one-page creative portfolio template. It is the perfect solution for Web Designers, Graphic Designers and any kind of creative people. Papel is built on Bootstrap v3.3.1 and packed with great features. It makes use of modern jQuery and CSS3 techniques to create superb eye-catching animations and make your visitors love you and your work. It is simple, modern and responsive. **Features** - Bootstrap 3.3.1 - Header and Portfolio Sliders - Fontawesome Icons - Responsive on all devices - Super Easy Configuration - and much more... **Changelog** V1.0 - Initial release V2.0 - Bootstrap 3.3.1 - Fixed logo alignment - Smooth animation **Credits** - Twitter Bootstrap - Animate.css - Eddie Lobanovskiy - VFS Digital Design - Fontawesome
